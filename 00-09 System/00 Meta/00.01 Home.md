# Vault Home

Welcome to your vault. This dashboard uses [Dataview](https://blacksmithgu.github.io/obsidian-dataview/) queries to surface active content.

---

## Active Projects

```dataview
TABLE status, tags
FROM "10-19 Projects/11 Active"
WHERE status = "active"
SORT file.name ASC
```

---

## Recent Notes

```dataview
TABLE file.mtime as "Modified"
FROM "20-29 Writing"
SORT file.mtime DESC
LIMIT 10
```

---

## Tracking

```dataview
LIST
FROM "40-49 Tracking"
SORT file.name ASC
```
